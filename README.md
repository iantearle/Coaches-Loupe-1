Coaches Loupe
========

Uploads screenshots to [Dribbble](http://dribbble.com) and [CloudApp](http://getcloudapp.com).

*This is still in beta, but give it a shot!*

[http://coachesloupe.com](http://coachesloupe.com "Coaches Loupe")
